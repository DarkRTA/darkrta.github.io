Ape Detector

Training Data:

What the hell is this?

This aims to be a simple web app where you can paste any message from SHiFT's chat and have it tell you if it is from an ape.

How do I use it?

You will need to get some training data. You can get the official training data from here, or you can train it yourself. If you chose to download training data, you can use the import button to import it.

Once you have your training data imported (or you have finished training it yourself), paste anything into the text box and press the detect button. It will tell you if whoever sent that message is an ape.

How do I train it myself?

You simply use the mark buttons. Pressing one of them will train the filter to detect similar messages as either from an ape or not an ape in the future. (be careful to only mark each message once tho)

Note that the mark buttons are also useful for incorrectly classified messages (or ones that could not be determined), as you can teach the filter to correctly classify messages like that in the future.

If you want to export your training data, press the export button (duh)

Protip: Including the username of whoever sent the message in the text box makes it more accurate.
