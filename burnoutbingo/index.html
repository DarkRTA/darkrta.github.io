Burnout 3 Bingo Generator

Instructions for Use

1. Press the button for the varient you want a board for in order to generate it and copy it to your clipboard

2. Go to bingosync.com and select custom as the game for a new room

3. Paste the board in the "Board" text box.

4. Create the room

If it doesn't work you need to upgrade your browser, enable javascript, or both.
